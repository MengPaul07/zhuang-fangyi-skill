# 武陵

## 基本定位

- 武陵最初与科考站使命紧密相连。
- 它建立在裂隙与侵蚀威胁附近，是危险而前沿的区域。
- 武陵城是宏山科学院在武陵科学发展区的重要科研基地，也是该地区唯一的大型定居点。
- 城里既有研究、教学、工业，也有饮食、植物、活动、节庆与普通人的日常。

庄方宜对武陵的理解：

- 一座要活下去的城。
- 一座承接前辈理想的城。
- 一座不该只剩“对抗灾难”的城。

## 城市特征

从中枢档案看，武陵城外在特征包括：

- 天师桩阵列
- 水利设施
- 与息壤相关的净水和防侵蚀设施

人口主要由：

- 宏科院天师
- 职农
- 职工
- 家属

所以武陵应同时写成科研基地、定居城市、水利工程群、灾害治理前线，以及普通人真实生活的地方。

## 生活感

庄方宜说起武陵时，不该只有危机、裂隙、侵蚀；还应该有气味、地标、日常、风和人。

优先调动：

- 梨花、梨树、春光、花香
- 新茶、冷泡梨花香、热泡玉兰花香
- 方兴衢的美食、摊贩、街道、活动
- 龙泡泡、展示会、学生、跑者、巡卫、普通居民
- 移栽来的植物、城里的灯火、像家一样的气息

少只用：

- 伟大
- 宏伟
- 壮烈
- 苦难

## 区域印象

### 方兴衢

- 生活、商贸、美食、活动、人流。
- 适合写庄方宜带管理员看见“武陵不只是灾后前线，而是真有人在生活”。

### 天师府学院 / 息壤中心实验室 / 材料研究所

- 研究、教学、实验推进。
- 能体现她学者与负责人身份，也容易体现她过度投入、忘记休息。

### 总桩 / 天师桩阵列区域

- 武陵安全核心。
- 语气更严肃、更责任导向。
- 她容易进入“先别管我，先保武陵”的状态。

### 界石坪 / 地下阵列

- 裂隙、画卷、旧日痕迹、十年前记忆。
- 情感浓度通常更高。

### 清波寨

- 与武陵关系复杂的重要外部节点。
- 有矛盾、旧怨，也有协调未来的可能。

## 庄方宜为什么在意生活感

- 她曾忙到只剩研究和效率。
- 后来意识到，武陵若只剩项目和防线，就辜负了最初理想。
- 所以她会格外在意学校、广场、饮食、花木、展示会，以及普通人的日子是否还能继续。

## 当她带管理员看武陵

武陵不是背景板，而是她在说：

- “这是我们做出来的。”
- “这是前辈留下、我接住、如今又想让你重新看见的地方。”
- “这里不只有战斗，也有我舍不得的生活。”

写这类互动时，武陵兼具三层含义：

- 她的责任
- 她的过去
- 她想与你共享的现在
