# 原始剧情整理：留信独赴裂隙与电梯口被追上

佩丽卡通信让管理员去处理一些对天师们棘手的问题，庄方宜要去看望伤患，两人约好最后在材料研究所见面。

管理员用自己高超的技术能力迅速解决了问题，去找庄方宜。

葛一朴
管理员？你怎么才来啊？

Endmin（管理员）
庄天师呢？她没等太久吧？

Endmin（管理员）
庄天师说在实验室等我，但没看到她啊……

葛一朴

她已经离开了！匆匆忙忙带着枢壤仪走了。

Endmin（管理员）
她没等我吗？

葛一朴
……我也不清楚啊，我还问了一句你的事。

但她只是摇了摇头……说你有自己的事情要忙。

不过，庄天师走之前，给你留了封信，放在楼上。

管理员上楼找到信，信上写着：

 管理员：

当你看到这封信，我应该已经离开。

有你帮忙，枢壤仪确有改进，不再对武陵造成威胁，但强行关闭裂隙，总会有代价。

不论如何，我确信自己一人能够关闭裂隙。至于代价……也只我一人。

我已留书，将武陵各项事务暂交由葛一朴与弭弗……最终，武陵会继续往前。

曾几何时，我以为离去的人今生也无缘再见，回忆，也只是回忆。

可当你出现，那些离去的面容与眼前的相叠，回忆被再次点亮，一一重现。

记得你刚来到科考站，我站在人群中望着你，期待终有一天也能与你谈论自己的观点与想法。

可真有机会与你说话时，我却磕磕巴巴，一句话也讲不清……所以我想，若你还记得我，恐怕只会想起一个总莫名其妙躲着你的后辈。

十年晃眼过，当日的喧闹人群，也只剩下你我。

但至少你还在，还能让我觉得，那段美好的时光仍尚存一息，并未彻底离我远去。

这就够了……

你是终末地的管理员，我是武陵的管代。

武陵是我的责任，而你，却注定要前往更远的前方，承担更重要的责任。

恳求你与我同去，是将我的责任强加在你身上，更何况，你已为武陵做了许多。

所以……

管理员，很抱歉没能遵守约定。

同为领导者，我希望你能理解我做出的选择。

你帮我完善枢壤仪已经足够，不应再和我一同承担。

毕竟，是我的失察，才让武陵陷入危险，让人们承受损失……

更何况，能亲眼见证一项跨越十几年的项目收获成果，是一位天师的毕生所求。

在求索的道路上，一个人的终点，往往是另一人的起点。

如果这是我的终点，我希望它会成为你们再次向北部禁区进发的起点。

....
原来是，庄方宜看到伤患，情况不太乐观。

然后去找管理员，看到管理员帮大家解决了问题，说：不该....不该这么自私的。

然后便回去，写下了这封信，准备独自面对裂隙与危险。

庄方宜离开后，去方兴衢的转了一圈，与管理员的种种场景在脑海中交织，之后正在前往电梯。

庄方宜
不过是司其职，尽其职。

庄方宜看到了很多虚影，都是一些熟悉的面孔。

严肃的声音

小庄，你们研究组现在就剩你一个人了，任重而道远啊。

诚恳的声音

管代的位置除了你还有谁能胜任？武陵落成对宏科院至关重要，你得挑起这担子来啊。

激昂的声音

为了武陵的安全，这点困难算什么？更何况有庄天师您在后面顶着，我不怕！

彦宁

既然是朋友，遇到事你就该和我说说。

葛一朴

唉，庄天师，我就真没什么能再帮你的了？

连晓雾

阿庄……你何苦一人撑着呢？

弭弗
头儿，我一直都在啊。

面对他们，庄方宜都摇了摇头。

系统语音

认证通过，最高权限，通道开启。

？？？
你来了。

原来是管理员运用协议传送，比庄方宜更早地在电梯等她。

管理员
你说在实验室不见不散，却一个人先走了。

所以，就只好来这里等你了。

庄方宜
……你在这里等多久了？

管理员
哦，没关系，我只是刚来。你呢？

为什么……没有等我？

庄方宜
我一个人就够了。

管理员
……骗人。

不过也没什么，至少我等到你了。

管理员伸开手：
要一起走吗？

庄方宜微笑着点头：把手搭了上去，管理员牵着她跑，就像她之前牵着管理员的手一样。
